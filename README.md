# 🏗️ Imperium Obras - Landing Page

Uma landing page moderna para uma empresa de arquitetura/construção, com design elegante, animações suaves e navegação dinâmica entre seções.

---

## ✨ Demonstração

Interface inspirada em layouts profissionais com:

* Card central com efeito de profundidade
* Ondas suaves no background
* Navegação dinâmica sem recarregar a página

---

## 🚀 Funcionalidades

* 🔄 Navegação entre seções sem reload (JavaScript)
* 🎬 Animações suaves de transição
* 🌊 Background com ondas SVG estilizadas
* 👥 Seção de equipe com cards personalizados
* 🏢 Seção de projetos
* 📞 Seção de contato
* 💻 Layout responsivo e moderno

---

## 🧱 Estrutura do Projeto

```
📁 projeto
 ├── index.html
```

---

## 🖱️ Como funciona

A navegação é feita com JavaScript:

* Cada seção ocupa o mesmo espaço (`position: absolute`)
* Apenas uma fica visível por vez (`.active`)
* Ao clicar no menu:

  * A seção atual desaparece
  * A nova aparece com animação

---

## 🎨 Tecnologias utilizadas

* HTML5
* CSS3 (Flexbox, Grid, animações)
* JavaScript (DOM manipulation)
